# WordPress Vibe Coding

Vibe coding for WordPress: a Hostinger-ready block theme project generated for a GitHub -> Hostinger Git deployment workflow.

## Project Goal

Build a polished Gutenberg-editable WordPress site through a versioned local workflow:

1. Develop locally with Codex.
2. Push code to GitHub.
3. Deploy from GitHub to Hostinger staging.
4. Review in WordPress.
5. Promote staging to production when ready.

## Current Theme

Theme path:

```text
wp-content/themes/vibe-aesthetic-theme
```

Recommended Hostinger deployment directory:

```text
public_html/wp-content/themes/vibe-aesthetic-theme
```

## First Milestone

Create the staging deployment path, connect this repository to Hostinger Git deployment, then activate the theme on the staging WordPress site.
